== Doc-as-Code Workshop

Dieses Readme führt durch eine Docs-as-Code Übung.

=== DTCW

zuerst installieren wir den docToolchain-Wrapper in der Umgebung.
Dazu müssen folgende Zeilen im Terminal eingegeben werden:

[code, bash]
----
curl -Lo dtcw doctoolchain.github.io/dtcw
chmod +x dtcw
----

Wir werden mit der Docker-Version arbeiten.
Deshalb ist kein

    ./dtcw local install java
    ./dtcw local install doctoolchain

notwendig.

Um auf generierte Files im Browser aus dem Github Workspace zugreifen zu können, starten wir einen static file server im Hintergrund:

[code, bash]
----
python -m http.server 8000 &
----

=== arc42 Template initialisieren

    ./dtcw docker installTemplate

installiert das arc42-Template unter src/docs .
Vorher wird docToolchain noch eine Standard-Konfiguration initialisieren.
